Proc gradboost - built the model - created the Astore file needed for scoring

The Astore file was created in previous steps using this type of code:

proc gradboost data=work.customer_churn_ml_train_final ntrees=100 maxdepth=5 assignmissing=useinsearch;
    input &char_vars / level=nominal;
    input &num_vars / level=interval;
    target &target / level=nominal;
    savestate rstore=gbstore;  /* creates ASTORE binary for scoring  */
    *code file='gbscore.sas';  /* creates 70,000 line SAS scoring program  */
run;

Going Further on Your Own: Registering a SAS Viya Workbench Model in SAS Viya Model Manager

Note: The following steps require access to SAS Viya Workbench and SAS Viya in your own environment. This task cannot be completed in the provided lab environment.

The results of the 03_Developing_a_Model.sasnb SAS notebook show that the Gradient Boosting model has the best fit statistics of all the models.

In the following cells, we will register the Gradient Boosting champion model in the SAS Viya Model Manager Repository.

Obtaining a SAS Viya Token

Open a terminal window (VS Code Menu > View > Terminal). Type and enter the following commands, and follow the prompts to setup the environment for uploading to Model Manager.

sas-viya profile init

For the prompts to this command, enter:

  • Service Endpoint: SAS Viya base url such as https://my.sas.viya.example.com/
  • Output type: json for example
  • ANSI colored output: y for example
sas-viya -k auth loginCode

Entering the previous command presents a link you open in a web browser, and sign in to obtain an authorization code.

Paste the obtained authorization code in the active terminal prompting for Code>.

sas-viya profile show -key access-token --details

Entering the previous command returns the full access-token to the terminal window.

Copy the token and place it after the word Bearer in the next SAS code cell below the %macro myTokenName().

Registering the Model in SAS Viya Model Manager

/* Download the astore file for proc registermodel to access */
proc astore;
   download rstore = work.gbstore
            store = "gbstore.sasast";
run;

/* Paste your SAS Viya token obtained in previous step */
%macro myTokenName() / secure;
"Bearer eyJqa3JkA...kbbp2fJnHw"
%mend;

/* Enter your SAS Viya URL for server */
/* A new Model Manager project called myProject with the gradient boosting model is created in SAS Viya */
proc registermodel
      name = "gbAstore"
      description = "GradBoost Astore Model"
      data = work.CUSTOMER_CHURN_ML_TRAIN_FINAL
      algorithm = GRADBOOST
      function = CLASSIFICATION
      server = "https://my.sas.viya.example.com"
      oauthtoken = "myTokenName"
      replace;
   project name="myProject" folder="myFolder";
   astoremodel store = "gbstore.sasast";
   target LostCustomer / level=binary event="1";
   assessment;
run;
